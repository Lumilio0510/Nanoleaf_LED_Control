# Nanoleaf Shape 画板设计文档

## 1. 产品定位

面向 Nanoleaf Shape 系列灯板用户的虚拟设计工具。用户可以在无限画布上，使用三种预设灯板类型（六边形、三角形、迷你三角形）自由拼接图案并设定颜色，保存设计方案并导出为图片。远期将接入生图大模型，支持上传墙面照片生成上墙效果图。

## 2. 技术选型

- **渲染引擎：** Konva.js (react-konva) — 基于 HTML5 Canvas，内置缩放/平移/拖拽/选中/导出图片功能
- **页面集成：** 在现有页面系统中新增一个 `canvas` 页面，与 control / skills / agent / settings 并列
- **数据存储：** JSON 文件保存在项目根目录 `designs/` 文件夹

## 5. 数据模型

```typescript
type PanelType = 'hexagon' | 'triangle' | 'mini-triangle'

interface PlacedPanel {
  id: string
  type: PanelType
  x: number        // 中心点 X
  y: number        // 中心点 Y
  rotation: number // 旋转角度（度）
  color: string    // Hex 颜色 '#ff0000'
  snappedTo: { panelId: string; connectionIndex: number } | null
}

interface CanvasDesign {
  id: string
  name: string
  panels: PlacedPanel[]
  createdAt: string
  updatedAt: string
}
```

## 6. 灯板几何定义

| 类型 | ShapeType | 边长 | 连接点数量 | 连接点位置 |
|------|-----------|------|-----------|-----------|
| 六边形 | 7 | 67 | 6 | 每条边中点 |
| 三角形 | 8 | 134 | 6 | 每条边两个三等分点 |
| 迷你三角形 | 9 | 67 | 3 | 每条边中点 |

几何约束：两个迷你三角形并排等于一个三角形的边长（67 × 2 = 134）。

## 7. 交互设计

### 工具模式

| 工具 | 快捷键 | 功能 |
|------|--------|------|
| 选择 | V | 点击选中、拖拽移动、框选、Shift 加选 |
| 放置六边形 | 1 | 点击画布放置，自动吸附 |
| 放置三角形 | 2 | 同上 |
| 放置迷你三角形 | 3 | 同上 |

### 放置吸附逻辑

1. 鼠标悬停时显示半透明预览
2. 检测附近（阈值 20px）已有灯板的连接点
3. 检测到吸附点 → 预览吸附到该位置
4. 点击放置，记录吸附关系
5. 没检测到吸附点 → 在鼠标位置自由放置

### 快捷键

- `Delete / Backspace` — 删除选中灯板
- `Ctrl+A` — 全选
- `Ctrl+Z` — 撤销上一步操作
- `空格 + 拖拽` — 平移画布
- `滚轮` — 缩放
- `G` — 重置视图

### 颜色交互

- 点击选中灯板 → 底部 ColorPanel 显示
- Shift+点击 / 框选 → 多选
- 颜色选择器（色环 + Hex 输入）+ 预设色板（12-20 种颜色）
- 批量调色：选中多块后调整颜色，同时应用到所有选中灯板

## 8. 文件管理

- 项目目录 `LED_Control/designs/` 下每个 JSON 文件对应一个方案
- 文件名 = 方案名称（自动处理非法字符）
- 支持操作：新建、打开、保存（Ctrl+S）、另存为、重命名、删除、复制
- 左侧方案列表，点击切换

## 9. 导出图片

- 导出格式：PNG（背景透明 / 白色可选）
- 调用 Konva `stage.toDataURL()` 生成
- 图片尺寸 = 所有灯板的最小包围矩形 + 边距
- 通过 Electron 原生对话框选择保存路径
